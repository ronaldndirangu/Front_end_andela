Who we are

As a web developer, you want to program in a language that is powerful, clean, mature, and well-documented, and one that is equipped with a great standard library and a huge selection of high-quality third-party packages. You also want a web framework for that language--one with a vibrant, helpful community of users and developers, and one that is designed to function smoothly as an integrated stack, but whose components are loosely coupled so that you can make substitutions if circumstances require. In short, you want Python and you want Django.

This is the framework we mostly us being a team python entuthiast with a keen interest in web development. We develop front ends usling angular and react javascript frameworks but for our back ends we use django, flask or pyramid depending on the requirement of the project

We hosts the web applications ourselves on our local servers and thus we end up saving money for you the client in hosting and maintenance services
